# ECC LAB ATTENDANCE

- We developed a Laboratory Attendance Program integrated with Elliptic Curve Cryptography (ECC) and ECDSA utilities.
- The system supports secure student registration, device binding, and QR-based attendance verification to help protect attendance validation from unauthorized tampering.
- It is designed to be efficient and practical for laboratory use while supporting browser-based portals and device-aware workflows.

---

## WHAT DOES IT DO?

- User Authentication
- Student Registration with device fingerprinting
- Session Creation and Management
- QR Code Generation and Validation
- Attendance Tracking with present/late status
- Reports and Exports
- Dashboard and API support

---

## PURPOSE AND USE CASE

This program is designed for the University of the Assumption Laboratories to automate attendance tracking, reduce manual errors, and provide secure and efficient attendance management. It helps staff manage lab sessions, verify student attendance via QR signatures, and analyze attendance data with confidence.
